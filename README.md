# Prioritera

A minimal priority tool. Drag items from your backlog into a stack to decide what to work on next — only the top item is in focus.

## Live
[GitHub pages](https://oskardrenske.github.io/prioritera/)

## Usage

Open `index.html` in a browser. No install required.

- **Click** a backlog item to push it onto the Todo stack
- **Click** the top Todo item to send it back to the backlog
- **Double-click** any item label to rename it

Items start with random JIRA-style IDs and reset on page reload.
